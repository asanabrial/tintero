Tintero is a minimal, markdown-first blog platform built with Next.js 16, Tailwind CSS, and a hand-rolled content engine.

## Philosophy

- **Content first**: your markdown files are the source of truth
- **Zero CMS**: no database, no external services
- **Fast by default**: statically generated, fully typed

## Technology

| Technology | Purpose |
|------------|---------|
| Next.js 16 | App router and static generation |
| Tailwind CSS | Styling |
| gray-matter | Frontmatter parsing |
| unified/remark/rehype | Markdown to HTML pipeline |
| Zod | Schema validation |
| Shiki | Syntax highlighting |

## Getting Started

1. Clone the repository
2. Run `bun install`
3. Add your posts to `content/posts/`
4. Run `bun run dev`

Tintero gets out of your way and lets your writing shine.
