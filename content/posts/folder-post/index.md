This is a folder-based post. The slug is derived from the folder name, not `index`.

## Benefits of Folder Posts

- Co-locate images with the post
- Organize related assets together
- Keep the content directory tidy

## Usage

Create a folder with an `index.md` inside `content/posts/my-post/`. The slug becomes `my-post`.
